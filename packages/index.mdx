Phrony publishes **open-source packages** so you can call the HTTP API with typed clients instead of wiring `fetch` yourself. Behavior matches the **[API Reference](/api-reference/index)**; packages add helpers, **`X-API-Key`** handling, and TypeScript types aligned with responses.

## Available packages

<CardGroup cols={2}>

<Card title="CLI — @phrony/cli" href="/packages/cli">
  Node **22.14+**. Lint manifests, sign in with OAuth, manage **agent** and **agent version**, **plan** / **apply**, and **diff** local YAML against the cloud.
</Card>

<Card title="TypeScript SDK — @phrony/sdk" href="/packages/typescript-sdk">
  Node **18+** or any runtime with **`fetch`** (including browsers and Deno). Agents, runs, conversation, stream (SSE), and the file library.
</Card>

</CardGroup>
