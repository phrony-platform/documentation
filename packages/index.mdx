Phrony publishes **open-source packages** so you can call the HTTP API with typed clients instead of hand-rolling `fetch` on every call. The API surface is the same as in the **[API Reference](/api-reference/index)**; packages add convenience, `X-API-Key` handling, and TypeScript types aligned with real responses.
