Most day-to-day work happens in the **Phrony dashboard** and the **[concept guides](/product/concepts/agent)**. Some teams also drive agents, triggers, and workspace objects from **their own systems** using Phrony’s **HTTP API**.

## Authentication

Automation typically uses **API keys** you create under **Settings** in the dashboard. Your organization’s deployment defines the exact headers or tokens.

## Reference docs

Interactive **OpenAPI** documentation will be linked here for your environment once it is published. Until then, use the dashboard for setup and rely on your admin or support channel for integration details.
