Most day-to-day work happens in the **Phrony dashboard** and the **[concept guides](/product/concepts/agent)**. Some teams also drive agents, triggers, and workspace objects from **their own systems** using Phrony’s **HTTP API**.

The default API base URL is **`https://api.phrony.com`**. Dedicated deployments may use a different host; your **API** trigger screen and workspace admin reflect the URL you should call.

## Authentication

Automation typically uses **API keys** you create under **Settings** in the dashboard. Send the raw key in the **`X-API-Key`** header unless your deployment documents otherwise.
