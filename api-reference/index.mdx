Phrony exposes a **JSON HTTP API** for running [agents](/concepts/agent) from your own backends and clients. Requests use `https` and, for authenticated routes, a workspace API key in the **`X-API-Key`** header.

## Base URL

| Environment | Base URL (example) |
| ----------- | ------------------ |
| Phrony Cloud | `https://api.phrony.com` |
| Self‑hosted or staging | As configured by your team (path layout under **`/v1`** is the same) |

There is **no** extra path prefix before the API version. Full example: `https://api.phrony.com/v1/agents/{agentId}/runs`.

## Quick examples

Set `PHRONY_API_KEY` and `AGENT_ID` from the Phrony dashboard. Use `runId` from the start response to poll. Each group shows the same **start a run, then get run** flow.

<CodeGroup>

```bash cURL
export PHRONY_API_BASE="${PHRONY_API_BASE:-https://api.phrony.com}"
export PHRONY_API_KEY="phk_..."
export AGENT_ID="00000000-0000-0000-0000-000000000000"

curl -sS -X POST "${PHRONY_API_BASE}/v1/agents/${AGENT_ID}/runs" \
  -H "Content-Type: application/json" \
  -H "X-API-Key: ${PHRONY_API_KEY}" \
  -d '{"input":{}}'

# Replace RUN_ID with the runId from the JSON body above
RUN_ID="00000000-0000-0000-0000-000000000001"
curl -sS "${PHRONY_API_BASE}/v1/runs/${RUN_ID}" \
  -H "X-API-Key: ${PHRONY_API_KEY}"
```

```typescript TypeScript
async function quickStart() {
  const baseUrl = process.env.PHRONY_API_BASE ?? "https://api.phrony.com";
  const apiKey = process.env.PHRONY_API_KEY ?? "";
  const agentId = process.env.AGENT_ID ?? "";
  const started = await fetch(`${baseUrl}/v1/agents/${agentId}/runs`, {
    method: "POST",
    headers: {
      "Content-Type": "application/json",
      "X-API-Key": apiKey,
    },
    body: JSON.stringify({ input: {} }),
  });
  if (!started.ok) {
    throw new Error(`start run failed: ${started.status} ${await started.text()}`);
  }
  const { runId } = (await started.json()) as { runId: string };
  const run = await fetch(`${baseUrl}/v1/runs/${runId}`, {
    headers: { "X-API-Key": apiKey },
  });
  console.log(await run.json());
}
void quickStart();
```

```python Python (stdlib)
import json
import os
import urllib.request

BASE = os.environ.get("PHRONY_API_BASE", "https://api.phrony.com").rstrip("/")
API_KEY = os.environ["PHRONY_API_KEY"]
AGENT_ID = os.environ["AGENT_ID"]


def post_json(path: str, body=None) -> dict:
    data = None if body is None else json.dumps(body).encode("utf-8")
    req = urllib.request.Request(
        f"{BASE}{path}",
        data=data,
        method="POST",
        headers={"Content-Type": "application/json", "X-API-Key": API_KEY},
    )
    with urllib.request.urlopen(req) as res:
        return json.load(res)


def get_json(path: str) -> dict:
    req = urllib.request.Request(
        f"{BASE}{path}",
        headers={"X-API-Key": API_KEY},
    )
    with urllib.request.urlopen(req) as res:
        return json.load(res)


started = post_json(f"/v1/agents/{AGENT_ID}/runs", {"input": {}})
run_id = started["runId"]
print(get_json(f"/v1/runs/{run_id}"))
```
